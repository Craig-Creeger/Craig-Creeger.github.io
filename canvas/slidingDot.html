<canvas> Animation

Basic animation using the Canvas element

Main Loop

The concept with Canvas element animation is simple – draw a scene, erase everything, draw it again with animated objects shifted by a pixel, repeat.

You will get better performance and smoother animation if the draw/erase cycles are synced to requestAnimationFrame().

Objects

Sliding Dot Example

See the code in “purpleDot.js”; it is highly documented and explained. Concepts demonstrated are:

  • Creating a context to the canvas.
  • Drawing a circle.
  • Wrapping Javascript in an anonymous function to keep the global namespace clean.
  • Simple Classes and object instances.
  • The animation loop.
  • Dynamically resizing the canvas based on the width of the browser window.

If your browser was capable of rendering the canvas tag you would be seeing a purple dot that moves back and forth horizontally.
