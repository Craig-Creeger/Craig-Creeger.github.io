Super Simple Image Rotation

Definition

If you need a really lightweight, easy-to-understand image slider, then this is it. You can place the div anywhere on the page. The images can rotate in a fixed or random order.

Limitations

  • This is not really a slider since the images cross-fade.
  • There are no start/stop controls, no progress indicator, no navigation.

Alternatives

If you are looking for something with more options, then AnythingSlider is pretty nice, but it takes up the full width of your viewport. The documentation blows, so good luck.

Adobe’s Spry UI has an image slider. I haven’t tried it, but it looks to have a lot of options, including the ability to pan and zoom.
