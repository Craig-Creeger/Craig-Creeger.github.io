Color Schemes

One of the first things you are going to do when creating a new graphic design is to choose a set of colors to use. By sticking with a standard, limited set (say, 5 or less colors) you will be able to help keep the site uniform.

Generate Colors From a Photo

This on-line tool will create a color palatte based on a photo that you upload. I think Photoshop can also do this. However, the easiest way to do this is with the iPhone/iPad app "Adobe Kuler".

Adobe's Kuler

This tool is built into Photoshop and Illustrator. You can also access it on the web or iPhone/iPad app.

Interactive Color Wheel

http://colorschemedesigner.com/

Adobe Illustrator has a similar function.
