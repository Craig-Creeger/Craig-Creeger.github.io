Checkboxes with Memory

These checkboxes will remember the last value, even if you close your browser and re-open it. Handy.





How it Works

The persistent memory is possible due to the localStorage API baked into HTML5. Clicking any of the checkboxes will execute a function that writes the value to localStorage. Opening the webpage will load any previously saved values (if there aren’t any, then it defaults to not checked).

How to Use

  1. Slap some checkboxes on your page; ensure they all have a unique id attribute and the name attribute is "todo".
  2. Copy/paste the following code to your page.
  3. Note: it is possible to use a different name for your checkboxes, in that case edit the checkName variable in the code.
<script type="text/javascript" src="../scripts/jquery.js"></script>
<script type="text/javascript">
	var checkName = 'todo';
 	$('input[type="checkbox"][name^="'+checkName+'"]').change(function() {
		if (window.localStorage) { //we use localStorage
 	        $('input[type="checkbox"][name^="'+checkName+'"]').each(function() {
 	            window.localStorage.setItem(this.id,Boolean(this.checked).toString());
 	        });
 	    }
 	});

 	$(document).ready(function() {
	   $('input[type="checkbox"][name^="'+checkName+'"]').each(function() {
 	        var thisChecked = window.localStorage[this.id];
 	        if (thisChecked===undefined||thisChecked=== 	'false') {
 	            this.checked = false;
 	        } else {
 	            this.checked = true;
 	        }
 	    });
 	});
</script>
