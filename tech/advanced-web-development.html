Evolution of the the Web

Web sites and applications are getting more complicated to create as the standards bodies, developers, and toolset makers pile on features (new CSS selectors and properties, new HTML tags, new Javascript methods and frameworks). At the same time that the code is getting more complicated, the number of devices and browser capabilities that run the web are increasing (iPads, mobile devices, various pixel ratios, varying degrees of support, etc).
