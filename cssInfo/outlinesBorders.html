Hidden and Hack

You can see that hidden and hack are a bit different. hidden as an actual CSS value and has the same meaning as none, except when it refers to table borders in cases where two cells share a border, and the table cells have collapsed borders (border-collapse:collapse;). The value hidden ensures that no border is drawn, since hidden takes precedence over all other border styles. If none had been used for one border in the cell, the border would still be drawn, as the adjacent cell’s border would take precedence. Notice how the hidden box is smaller than the others; this is probably not what you want, hence my hack.

Hack is not a CSS value; it is a technique I use when I need a border’s space to exist, but not be visible. I do this by setting the border color to transparent. i.e. <div style="border: solid thick transparent"></div>

I do this when my links are styled as a button and a mouse over (a:hover) selector toggles the display of the border.

Shorthand Notation

These three properties

border-style:solid;
border-width:thin;
border-color:black;

can be combined into the shorthand notation as:

border:solid thin black;
