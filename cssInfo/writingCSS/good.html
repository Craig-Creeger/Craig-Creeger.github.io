Additional Info

An example of a poorly constructed web page has been created so that you can compare the differences and learn from others’ mistakes.

This is a simple layout that contains the core features of most websites. We have a header, an area for navigation, a “main” content area, sidebar, and a footer. Take a look at the source code of this page and you will notice that:

  • Only two <div> tags are used and they are both absolutely necessary. The first one, <div id="wrap">, is used to center this content; admittedly it is not semantic and does add extra code to the HTML, but necessary, none-the-less. The second <div role="main"> is semanatic (an ARIA marker for the main content of the web page).
  • <header> and <footer> tags are used for both semantic meaning and to target specific CSS styles to them.
  • <header> tags can be used multiple times. They are styled with the diagonal grid.
  • A class, “pushRight” is reused for the aside and logo.
