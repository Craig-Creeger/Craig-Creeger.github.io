3D Effects Using CSS

Perspective

Think of DOM objects as normally existing on a plane that is the same as the surface of your monitor. Push that item away from you (z-axis) and it gets smaller. Tip the top away from you (x-axis rotation) and it will appear smaller while at the same time the bottom will appear to get larger.

This is an unadulturated square.

transform: translateZ(-300px);

transform: rotateX(45deg);

For any of this to work, you need to tell CSS that you want to render items in a 3D space. You can either create a container where all items inside it share the same vanishing point. You do that with the perspective property. Ex:

.space3D {
	perspective: 700px;
}

Alternatively, you can use the perspective() function if you need to transform only one DOM element independent of any others. Ex:

transform: perspective(700px) rotateY(35deg);
