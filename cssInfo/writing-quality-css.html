Writing Quality CSS

People new to writing HTML and CSS often make it way more complex than it needs to be. If you inhert a project that has unweildly CSS, there is tool called UnCSS that will identify CSS styles that can be removed. It can be combined with Grunt to automate the build process.

Another tool, uCSS, is better if you want to manually remove the bloat.
