Rounded corners

Rounded corners on your boxes can be achieved through CSS via the property border-radius.

-moz-border-radius: 15px;
-webkit-border-radius: 15px;
border-radius: 15px;

It is possible to tweak each corner differently by specify each corner similarly to the way individual margins are specified. Ex.:

-moz-border-radius: 5px 16px 0px 30px;
-webkit-border-radius: 5px 16px 0px 30px;
border-radius: 5px 16px 0px 30px;

The sequence is top-left, top-right, bottom-right, bottom-left.

You can use an on-line tool to write this code for you.

CSS3, Please! has a good interactive way of testing this property.
