Bluebird Promise Library

Advantages

jQuery comes with Promises (via the Deferred object), but Bluebird claims to be faster, is focused on real-world use-cases, and works everywhere.

Usage

After including the library in your page (<script src="bluebird.min.js"></script>), you could create a Promise like this:

function getRandomNumber(max) {
	return new Promise(function (resolve, reject) {
		var rand = Math.random() * max;
		rand = Math.floor(rand);
		if (rand > max * 0.4) {
			resolve(rand);
		} else {
			reject(rand);
		}
	});
}
getRandomNumber(100)
.then(function(data) {
	console.log("SUCCESS: " + data);
}).catch(function(data) {
	console.log("FAIL   : " + data);
});
