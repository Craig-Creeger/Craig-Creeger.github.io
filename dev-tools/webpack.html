Webpack

Overview

Webpack can often replace Grunt or Gulp because it can transpile Javascript and CSS, and intelligently concatenate fields together. It supports various module systems such as AMD, CommonJS, ES6, and Angular.

The Webpack How-To Guide is a great resource for getting started with it.

The official Webpack documentation is known for being crappy, however several tutorials are listed on that site.
