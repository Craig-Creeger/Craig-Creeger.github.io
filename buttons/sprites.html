Basic Sprite Buttons

Definition

As CSS-based Sprite button is a technique where a button has a graphical (jpg, png, or gif) background. Youâ€™ll notice the strip of buttons below do not contain any text; the text comes from your HTML. Because all the buttons are in a single file, we reduce the number of HTTP Requests, which decreases the time needed to load the web page.

The magic that makes this technique work is CSS property background-position.

More Information

How to create CSS-based Sprite buttons

I'm not sure if it's true or not, but I've read that your browser can render the image swaps faster when the buttons are layed out horizontally like this (single image) shown here:

My light gray buttons are a "disabled" state. In other words, the user can still see them, but they are non-operational.

1. Place all your graphics in one PNG file.

2. Place the button on your page with an <input type="button"> or <a> tag.

Test Me <— This was created as <a class="wideBtn" href="javascript:void(0);">Test Me</a>

<— This was created as <button class="wideButton" type="button">Test Me</button>

<— This was created as <input class="wideButton" type="button" value="Test Me">

3. Write the CSS. Shown here are two different ways to write your CSS. The first is if your buttons are created with the <a> tag and the second is for <input> or <button> tags.

/*    SPRITE BUTTONS (for anchor tags in the HTML) */
a.wideBtn { /* This will apply to both unvisited and visited links */
	background-image: url(buttonsSeaRay.png);
	background-repeat: no-repeat;
	width: 185px;
	height: 33px;
	display: inline-block;
	text-decoration:none;
	text-align:center;
	background-position: -240px top;
	vertical-align: middle;
	line-height:33px;
	color:black;
}
a.wideBtn:hover, a.wideBtn:focus {
	background-position:-425px 0;
	font-weight:bold;
}
a.wideBtn.disabled {
	color:white;
	background-position:-610px 0;
	font-weight:normal;
	cursor:default;
}


/*    SPRITE BUTTONS (for button or input tags in the HTML) */
.wideButton {
	background-image: url(buttonsSeaRay.png);
	background-repeat: no-repeat;
	width: 185px;
	height: 33px;
	display: inline-block;
	background-position: -240px top;
	color:black;
	border:none;
	font-size:1rem;
	background-color:transparent;
	cursor:pointer;
}
.wideButton:hover, .wideButton:focus {
	background-position:-425px 0;
	font-weight:bold;
}
.wideButton.disabled {
	color:white;
	background-position:-610px 0;
	font-weight:normal;
	cursor: default;
}

Multi-line Buttons

The above code works great when you are vertically centering a single line of text in each button, but if there could possibly be multiple lines of text, then modify the CSS by changing the display property.

a.twoRowButton {
	vertical-align:middle;
	display:table-cell;
}
