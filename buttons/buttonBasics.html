Button Basics

Button vs. Input vs. Anchor

You want to put a a rollover button on your page, but which method should you choose? First ask yourself, what should happen when you click the button? If a new page will load in the window, then that is the same behavior as a hyperlink so I would argue that symantically it ought to be an <a> tag and it will carry more weight from an SEO standpoint.

If the button will perform a client-side script and stay on the same page, then a <button> or <input> tag would describe that function better.

<button type="button"> tag

Example:

<button class="wideButton" type="button">Click Me</button>

Notice that the text that appears on the button is surrounded by the tag. Compare that to the <input> tag where it is a value attribute.

<input type="button"> tag

Example:

<input class="wideButton" type="button" value="Click Me">

Inside a form, it may be better to use an input element for compatibility reasons due to the way various browsers handle form submission.

<a> tag

Example:

<a class="wideButton" href="#">Click Me</a>

Make 'em Pretty

I tend to use the <button> tag for on-page actions and <a class="button"> tags for navigation or off-page actions, however I like them to look the same.

Examples:

This is button 1   This is an anchor

I have written instructions for styling all three button types using CSS Sprites. There are more advanced techniques described in on the Rollover Buttons page.

button, .button1 { /* Style the <button> element, and optionally make <a> or <input submit|button> elements look the same. */
	text-decoration:none;
	color:#7a0019; /* maroon */
	display:inline-block;
	padding:2px .7em;
	background-color:#ffcc33; /* gold */
	border-radius:6px;
	border:medium solid #7a0019;
	font-size:.875rem;
	font-weight:normal;
	line-height:inherit;
	cursor:pointer;
	-webkit-appearance: none;
	font-family:Verdana, Geneva, sans-serif;
}

button, .button2 { /* Style the <button> element, and optionally make <a> or <input submit|button> elements look the same. */
	text-decoration:none;
	color:black;
	display:inline-block;
	padding:2px .7em;
	background-color:whitesmoke;
	border:thin solid dimgray;
	font-size:.875rem;
	font-weight:normal;
	line-height:inherit;
	cursor:pointer;
	-webkit-appearance: none;
	font-family: Arial, Helvetica, Verdana, Geneva, sans-serif;
	box-shadow:2px 3px 8px darkgray;
}
