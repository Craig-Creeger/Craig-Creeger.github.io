Definition

Textures are large, unobtrusive graphic elements that are typically used as a background. They are an image with no subject - for example a brick wall, grass, fabric, paper, etc. This page is using a marble wall as the background texture.

Some sites with good, high-quality textures are:

  • Skytech Geek has some examples, including free downloads of some popular textures.
  • Graphics Fuel website.
  • Render Textures has some really great repeating pattern wood, grass, bricks, etc.

 

How to Create Your Own Repeating Background Texture

Using Photoshop, this little bit of tiling magic is done with the Filter/Other/Offset command. The Smashing Photoshop CS5 book has a decent tutorial for creating a repeating background, but they failed to mention a few key items to make your repeating background perfect.

  1. Save this image of ocean waves to get started.
  2. Apply the Offset filter.
  3. Stamp or Heal the hard boundaries. Stay away from the outside edges of your image. Those pixels are needed to seamlessly tile.
  4. Apply the Offset filter one more time and look for any defects; clean them up if necessary.
