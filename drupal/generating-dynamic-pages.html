Generating Dynamic Pages

For pages where the content is built on the fly (vs. Nodes or Views), you can programmatically built it in PHP, or even Javascript.

Boilerplate

/**
 * Implements hook_menu.
 */
function creeger_menu() {
	$items = array();
	$items['creeger/mypage'] = array(
		'title'=>'My Sample Page',
		'description'=>'This is just a test for now',
		'page callback'=>'_build_mypage', //name of a function to call
		'type'=>MENU_NORMAL_ITEM,
		'access callback'=>TRUE, //effectively removes security - open to anonymous
	);
	return $items;
}

function _build_mypage() {
	return '<p>This is a sample page.</p>';
}

Navigate your browser to example-site.com/creeger/mypage and you will see the text “This is a sample page.”

Function _build_mypage() will certainly be much more complicated; probably generating a Render Array that is themed by a different function yet.
