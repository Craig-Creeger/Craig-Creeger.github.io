How to Fix It

This information was based on a Smashing Magazine’s “Setting Up Photoshop for Web, App, and iPhone Development.” Another article, Photoshop CS5 Color Management, goes into much more detail and explains why/how color management works.

  1. Assign the “Don’t Color Manage This Document” profile. For new documents, this can be done in the new document dialog box.

    For existing images, choose Edit → Assign Profile...

  2. Set your Color Settings to “Monitor Color”. From the menu, choose Edit → Color Settings...

    The important piece you are setting here is the RGB Working Space gets set to your Monitor RGB color space.

  3. Ensure View → Proof Colors is turned off (unchecked).
  4. When saving files with Save for Web & Devices, ensure that Convert to sRGB is turned off. If you're saving a JPEG file, then also turn off Embed Color Profile.
  5. Lastly, you may find that colors are still shifting if the JPEG compression is very low (around 15). Try 30 or higher if needed.
